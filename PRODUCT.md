# XActions — Product context

## Product
Browser extension (Chrome MV3) for X/Twitter growth automation without API fees. Primary surface: popup on the user's logged-in x.com tab.

## Users
Solo creators / growth operators who already use X daily. They want the AI to **diagnose the account and run a plan**, not chat forever.

## Primary job
**Analyze → Playbook → Execute** (Strategist). Manual automations and chat are secondary.

## Constraints
- Popup ~390×600px, dense product UI
- Dark tool aesthetic (X-adjacent blues), system fonts
- Vietnamese UX copy by default on this machine
- No backend required for daily use

## Success
User can open extension → see connection + identity → run analysis → review steps → execute safely without hunting through Auto cards.
