# XActions Extension — Design system

## Register
Product tool UI (restrained). Design serves the task.

## Tokens
- BG: near-black `#050608`, elevated `#0f1216`, card `#14181f`
- Accent: X blue `#1d9bf0` (primary actions only)
- Success `#00c48c` · Warning `#f5c518` · Danger `#f4212e`
- Radius: 10–14px cards, full pill for chips/tabs
- Type: system-ui stack, 11–14px product scale

## Hierarchy (popup)
1. Header: brand + Live/Offline + emergency controls
2. Compact identity strip (account)
3. Tabs: **Plan** (default) → Auto → Log → Settings
4. Plan tab stages: Setup → Analyze → Playbook → Run · Chat collapsed

## Motion
150–220ms state only. No page choreography.
